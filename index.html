SportEventsPlanner

App to organize sport events

View project on GitHub

Overview

SportEventsPlanner is a modern Single Page Application which allows users to publish and join to sports meetings organized by friends. Users can create an account, publish their sport event, and join an organized event. App is allowing the user to create and delete a sport event.

Key features

  • SPA design and architecture
  • User Interface based on Bootstrap
  • Google Firebase implementation
  • Both traditional and social-media based user authentication
  • Ensuring application responsiveness - users of mobile and desktop devices
